12 - Object Interface

Interfaces are supposed to only check functions, but type_czech.objectInterface() checks for other types as well if so wished.

This code checks for the methods functionOne(), functionTwo(), and the number property aNumber being in an object. 
TWO_FUNC_INTERFACE = { functionOne: 'Function', 
                       functionTwo: 'Function', 
                       aNumber: 'Number'}

an_issue = type_czech.objectInterface(some_object, TWO_FUNC_INTERFACE);
In the examples to the left, the first person() function parameter is checked to see if it implements the showPerson() function, if not it fails the interface.

Type-Czech is set to throws exceptions in this example to cut down on the number of error messages.
