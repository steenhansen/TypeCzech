27 - Typed and Empty Union Checks with Optionals

Overloaded functions with more than one type of parameter signature, and extra typeless optional arguments, can be type checked. 
type_czech.validUnionExtras(arguments, type_signatures)
Overloaded functions with more than one type of parameter signature, and extra typeless optional arguments, can be checked against empty valueless values. 
type_czech.valuelessUnionExtras(arguments, empty_signatures)
