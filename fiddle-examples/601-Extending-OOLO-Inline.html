TypeCzech's general example of an author where:
  • First controls an author's first name
  • Last handles an author's first and last name
  • Full deals with an author's salutation, first, and last name
  • Safely removable supporting TypeCzech test code begins with /**/

OOLO create()s are used for inheritance:
  • All JavaScript is in one file, no code imports
  • Production class declarations must interleave with TypeCzech calls to have the PRE and POST checking functions be correctly linked to before use

Note this OOLO version kills the instanceof operator, use instanceOf() instead.
