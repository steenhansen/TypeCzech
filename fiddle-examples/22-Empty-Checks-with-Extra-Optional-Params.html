22 - Empty Checks with Extra Optional Parameters

Extra optional parameter values are not checked for being empty or valueless, they are just ignored. As long as the correct declared shape of empty states is not violated, any additional elective parameters will be overlooked when using type_czech.valuelessExtras(arguments, expected_empty).

There is no way of specifying optional keys, and optional array elements.

Objects specify their mandatory keys and types, whilst arrays just check the begining elements for types, and ignore the rest undescribed elements.
