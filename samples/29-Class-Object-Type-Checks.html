29 - Class Object Type Checks

Class objects can have their types checked. The below function shows the object inheritance lineage. 
type_czech.objectPrototypes(an_object)
This is how to get the final object type. 
type_czech.objectType(an_object)
Finally, you can interogate an object to see if it is an instance of a class. 
type_czech.objectIsA(an_object)
