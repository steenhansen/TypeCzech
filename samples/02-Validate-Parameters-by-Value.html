02 - Validate Parameters by Value

Programmatical checks can be performed before your functions are executed with removable PRE testing functions. Scalar arguments cannot be affected, but object and array reference parameters can, so care must be taken not to mutate objects and arrays inside the Type-Czech checking functions.

Errors are delivered by non-empty string values, thus a return value of '' indicates all is well. No return value from PRE checking functions is also indicative of no error.
