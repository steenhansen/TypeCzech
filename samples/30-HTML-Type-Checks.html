30 - HTML Element Type Check

HTML elements can have their types checked. 
html_type = type_czech.objectType(html_element)

HTML elemnts can also be queried for their prototypes chain. 
html_prototypes = type_czech.objectPrototypes(an_html_element)

And finally, HTML elements can also be asked their object type. 
isa_HTMLElement = type_czech.objectIsA(js_div, 'HTMLElement')
