17 - Arrays with One Type

Arrays with one type of element can be easily checked for compliance. 

type_czech.valid(arguments, 'String')        // one string

type_czech.valid(arguments, ['String'])  // array of any number of strings 0 ...

type_czech.valid(arguments, ['String', 'String']) // array of exactly 2 strings

type_czech.valid(arguments, [ ['String'] ]) // arrays of string arrays
