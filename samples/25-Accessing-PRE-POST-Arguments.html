25 - PRE and POST Arguments

The arguments object serves as a useful default parameter variable, thus normal functions are much easier to use with Type-Czech than arrow functions which lack the arguments object. Both standard parameters and the argument object can be used in PRE and POST checking functions, such as this page's examples.

Checking for exactly one string parameter only via parameter name. 
type_czech.valid(single_parameter, 'String')
Seeking exactly one string parameter only via arguments object. 
type_czech.valid(arguments, 'String')
Validating an array of strings via parameter names with a length of exactly 3. 
type_czech.valid([first_str, second_str, third_str], ['String'])
Looking for an array of strings with any number of element via arguments object. 
type_czech.valid(arguments, ['String'])
So an array of exactly one element, must be checked for a length of one also as there is no canonical represention of an array of one element, only a single element or an array of 0 or more elements. 
string_array_issue = type_czech.valid(arguments, ['String'])
if (string_array_issue) return string_array_issue
if (arguments.length !== 1) return 'Error, array does not have one element'
return ''
