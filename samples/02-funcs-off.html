02 Turning TypeCzech Parameter checking Off
This page does not include the TypeCzech library.

To stop parameter checking with TypeCzech in production you can do any of the following:
  • Do not include the <script src="TypeCzech.js"></script> library in the file
  • Do not declare the checking functions PERSON() and SHOW()
  • Do not call type_czech.precedeCheck(Person, PERSON)
  • type_czech = TypeCzech('NO-CHECKING') if library is still included
When the TypeCzech library may or may not be included, then this section of code is needed so the user functions does not get affected by its presence or absence: 
  if (typeof TypeCzech === 'function'){
    type_czech = TypeCzech('LOG-ERRORS')   
  }else{
    type_czech = { precedeCheck : do_nothing => do_nothing }    
  }
When the checking function PERSON() may or may not be defined, then this section of code is needed so the program does not get affected by its existence or absence either: 
  PERSON = (typeof PERSON === 'undefined') ? undefined : PERSON
Note that there is a very small performance penalty for keeping the checking functions as is when the TypeCzech library is not present. The Console Output shows that the Person() has not changed.
