36 - Turn-On, Turn-Off, and Show TypeCzech Counting

TypeCzech can stop and start checking parameters with:
  • type_czech.checkBegin()
  • type_czech.checkEnd()
TypeCzech defaults to checking parameters when initilized. Show the number of errors and checks found with:
  • type_czech.failureTally()
  • type_czech.checkTally()
To get the errors/checks ratio use type_czech.failureRatio().
