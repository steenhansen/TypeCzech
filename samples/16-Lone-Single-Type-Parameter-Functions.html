16 - Lone Single Type Parameter Functions

To check for exactly one parameter, as with PRE_exactly1Str(), the arguments length has to be checked, otherwise the extra parameters are just ignored.

Undefined values are printed as 'un-defined' in error messages to distinguish the undefined value from the 'undefined' string which look the same when printed in the console.

So when only checking a single lone parameter, leave off the array brackets - [ ], to verfiy that there is only one number being passed into the function. 

type_czech.valid(arguments, 'Number')              // one number

type_czech.valid(arguments, ['Number', 'Number'])  // two numbers

type_czech.valid(arguments, ['Date', 'String'])    // two values
