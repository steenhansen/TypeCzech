TypeCzech's general example of an author where:
  • First controls an author's first name
  • Last handles an author's first and last name

Extended classes are used for inheritance:
  • When TypeCzech is turned off no JavaScript code is imported
  • With TypeCzech turned on, the supporting test code is imported:
    • TypeCzech.js
    • TypeCzech_First.js
    • TypeCzech_Last
    • TypeCzech_Full.js
