21 - Type Check with Extra Optional Parameters

Extra optional parameter values are not checked for correct type, they are just ignored. As long as the correct declared type shape is not violated, any additional elective parameters will be overlooked when using type_czech.validExtras(arguments, expected_types).

There is no way of specifying optional keys, and optional array elements.

Objects specify their mandatory keys and types, whilst arrays just check the begining elements for types, and ignore the rest undescribed elements.
