36 - Turn-On, Turn-Off, and Show TypeCzech Counts

TypeCzech can stop and start checking parameters with:
  • enableTests()
  • disableTests()
TypeCzech defaults to checking parameters when initilized. Show the number of errors and checks found with:
  • countFails()
  • countTally()
Once TypeCzech is logging or throwing errors from checking functions, then the only way to stop the test exceptions or console logging is to is use the type_czech.disableTests() function. From then on all PRE and POST checking functions will not report errors, but will still consume a non-trivial amount of resources. So never use type_czech.disableTests() in production.

Note that enableTests() cannot start TypeCzech function checking from a non-active TypeCzech instance. Also enableTests() resets test counting to 0.
