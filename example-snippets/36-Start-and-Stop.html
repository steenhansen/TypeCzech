36 — Turn-On, Turn-Off, and Show TypeCzech Counts

TypeCzech can stop and start checking parameters with:
  • type_czech.enableChecks()
  • type_czech.disableChecks()
TypeCzech defaults to checking parameters when initilized. Show the number of errors and checks found with:
  • type_czech.countFails()
  • type_czech.countTally()

Once TypeCzech is logging or throwing errors from checking functions, then the only way to stop the test exceptions or console logging is to is use the type_czech.disableChecks() function. From then on all PRE and POST checking functions will not report errors, but will still consume a non-trivial amount of resources. So never use type_czech.disableChecks() in production.

Note that enableChecks() cannot start TypeCzech function checking from a non-active TypeCzech instance.
