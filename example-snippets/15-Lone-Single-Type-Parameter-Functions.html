15 – Lone Single Type Parameter Functions

To check for exactly one parameter, as with PRE_check_exactly1Str(), the arguments length has to be checked, otherwise the extra parameters are just ignored. Note the parameter names string_1 and string_2 are place holders and the arguments object is used instead.

So when only checking a single lone parameter, leave off the array brackets - [ ], to verfiy that there is only one number being passed into the function. 
type_czech.check_type(arguments, 'number')              // one number
type_czech.check_type(arguments, ['number', 'number'])  // two numbers
type_czech.check_type(arguments, ['date', 'string'])    // two values
