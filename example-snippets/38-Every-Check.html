38 — Every Check

This page exists to highlight each possible checking function that executes within a PRE or POST test routine. Each of the allTypes() checking function calls, from A to J, can be executed in the console causing at least one error. Short cuts as, 's' for 'string', and 'ER' for 'EMPTY-ERROR', save space.

No error for the below function call when tested in the current page's console. 
allTypes('John', 'Winston', 'Lennon')
A. check_emptyExtra(arguments, expected_emptys) 
allTypes('', '', '')  // param_1 is empty
B. check_typeExtra(arguments, expected_types) 
allTypes(7,11)  // not strings but numbers
C. check_emptyEither(arguments, expected_emptys) 
allTypes('Ringo', '', '')  // matched neither empty
D. check_typeEither(arguments, expected_types) 
allTypes('Lennon', '', 17) // param_3 not str or obj
E. check_empty(arguments, expected_emptys)
