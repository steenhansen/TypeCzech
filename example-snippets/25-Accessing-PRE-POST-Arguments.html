25 — PRE and POST Arguments

The arguments object serves as a useful all purpose array-like parameter variable, thus normal functions are much easier to use with TypeCzech than arrow functions which lack the arguments object. Both standard parameters and the argument object can be used in PRE and POST checking functions, such as this page's examples.

Checking for exactly one string parameter only via parameter name. 
type_czech.check_type(single_parameter, 'string')
Seeking exactly one string parameter only via arguments object. 
type_czech.check_type(arguments, 'string')
Validating an array of strings via parameter names with a length of exactly 3. 
type_czech.check_type([first_str, second_str, third_str], ['string'])
Looking for an array of strings with any number of element via arguments object. 
type_czech.check_typeVariadic(arguments, ['string'])
Two Similar Checks
Here check_type(arguments, ['string']) verifies that one array of strings is the sole parameter with a parameter of ( ['cat', 'dog', 'bird'] ). 
function PRE_check_arrayOfStrs(){
  return type_czech.check_type(arguments, ['string'])
}

arrayOfStrs = type_czech.linkUp(arrayOfStrs, PRE_check_arrayOfStrs) 

function arrayOfStrs(){
  console.log('arrayOfStrs()', arguments)
}
arrayOfStrs( [ 'cat', 'dog', 'bird' ] )
Below shows check_typeVariadic(arguments, ['string']) testing that one or more strings are separately passed as one or more variadic parameters with parameters of ( 'cat', 'dog', 'bird' ), not an array as above. 
function PRE_check_variadicStrs(){
  return type_czech.check_typeVariadic(arguments, ['string'])
}
  
variadicStrs = type_czech.linkUp(variadicStrs, PRE_check_variadicStrs) 
  
function variadicStrs(){
  console.log('variadicStrs()', arguments)
}

variadicStrs( 'cat', 'dog', 'bird' )
