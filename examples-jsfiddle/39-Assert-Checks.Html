39 — Third Party Result Checks

When working with libraries we cannot link into functions with TypeCzech.linkUp() to check function parameters and return values. But, we can still manually check function results with TypeCzech.assert_check(check_error, func_name, param_value) as is done to the left. Below are two short examples, one throws an exception. 

type_czech = TypeCzech('THROW-EXCEPTIONS')       // Error, expected Object
return_object_value = [1,2,3]
type_error = type_czech.check_type(return_object_value, "object")
type_czech.assert_check(type_error, 'ObjectCheck',return_object_value)
Note that if checking is turned off, via TypeCzech.disableChecks(), or with TypeCzech('NO-CHECKING') then the checking functions, like check_type() and assert_check() return immediately without doing any work. This means that the below code does no testing, and immediately returns. 
type_czech = TypeCzech('NO-CHECKING')           // Error ignored no checking
return_object_value = [1,2,3]
type_error = type_czech.check_type(return_object_value, "object")
type_czech.assert_check(type_error, 'ObjectCheck',return_object_value)
