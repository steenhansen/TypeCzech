TypeCzech

TypeCzech is a detachable run-time type checking JavaScript library that can be toggled on and off. Include one file in your web page or in your Node.js file to allow you to type check function parameters and results. Errors can be set to throw exceptions or just output to the Console. The motivation is to verify function parameters before execution, and function results after completion with PRE_check() and POST_check() funtions.

The shortest possible example on a live JSFiddle.
