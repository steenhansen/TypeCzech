403 - ClassFree Constructors Production

TypeCzech's general example of an author where:
  • First controls an author's first name
  • Last handles an author's first and last name
  • Full deals with an author's salutation, first, and last name
  • Safely removable supporting TypeCzech test code begins with /**/
ClassFree constructors are used for inheritance:
  • When TypeCzech is turned off no JavaScript code is imported
  • With TypeCzech turned on, the supporting test code is imported:
    • TypeCzech.js
    • TypeCzech_First.js
    • TypeCzech_Last
    • TypeCzech_Full.js



  • Production classFree declarations must interleave with TypeCzech calls to have the PRE and POST checking functions be correctly linked before use
