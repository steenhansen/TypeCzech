11 — Object Properties

Object structure can be checked with type_czech.check_type().

This code checks that the properties of funcOne and funcTwo are both functions. And the property a_number must be a Number type. Missing or extra properties will cause errors. 
FUNC_NUM_FUNC_STRUCTURE = { funcOne: 'Function', 
                            a_number: 'Number',
                            funcTwo: 'Function' }
func_num_func_obj = {funcOne: x=>x, a_number:12, funcTwo: y=>y}
check_error = type_czech.check_type(func_num_func_obj, FUNC_NUM_FUNC_STRUCTURE)
The below example shows how to ignore some properties, while still type checking at least one property. Here we only ensure that the property funcOne is a Function, the extra property my_string is ignored. 
FUNC_ONE_ONLY_STRUCT = { funcOne: 'Function'}
func_num_func_obj = {funcOne: x=>x, my_string:'some-string'}
check_error = type_czech.check_typeExtra(func_num_func_obj, FUNC_ONE_ONLY_STRUCT)
In the examples to the left, the first person() function parameter is checked to see if it implements the showPerson() function, if not it fails the check_type() call.

Type-Czech is set to throw exceptions in this example.
