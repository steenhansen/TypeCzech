06 — Validate Parameters and Results

One use for the type_czech.link(your_function, before_test, after_test) construct is to help with debugging and testing without changing the actual source code. The before and after conditions of functions can be viewed without asserts actually littering the functions.

Note that any function names can be used, not just PRE_check_function_name() and POST_check_function_name(), which are used to lessen confusion. 

function BEFORE_Person(){}

function Person(){}

function person_after() {}

Person = type_czech.link(Person, BEFORE_Person, person_after)
